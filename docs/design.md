# SemiNexus CLI 设计文档

## 1. 核心定位

**SemiNexus CLI** 是一个纯命令行的 Agent 能力中转工具，不依赖 Web UI，直接通过命令行管理 AI Agent 的 Skills、MCP、Agents 等能力。

## 2. 整体架构

```
┌─────────────────────────────────────────────────────────────────┐
│                         用户终端                                  │
│                                                                 │
│   $ semi-nexus <command> [options]                              │
│                                                                 │
└────────────────────────────┬────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│                      SemiNexus CLI                               │
│                                                                 │
│   ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐           │
│   │  login  │  │ search  │  │install  │  │  sync   │  ...      │
│   └────┬────┘  └────┬────┘  └────┬────┘  └────┬────┘           │
│        │            │            │            │                  │
│        └────────────┴────────────┴────────────┘                  │
│                         │                                        │
│                    ┌────┴────┐                                   │
│                    │  Config │  (~/.semi-nexus/config.yaml)       │
│                    └────┬────┘                                   │
│                         │                                        │
└─────────────────────────┼────────────────────────────────────────┘
                          │
                          ▼ (可选)
┌─────────────────────────────────────────────────────────────────┐
│                    SemiNexus Registry                           │
│              (本地文件或远程服务，可选)                           │
└─────────────────────────────────────────────────────────────────┘
```

## 3. 命令列表

| 命令 | 功能 |
|------|------|
| `init` | 初始化配置目录 |
| `login` | 登录认证 |
| `logout` | 登出 |
| `search` | 搜索能力 |
| `subscribe` | 订阅能力 |
| `install` | 安装能力 |
| `sync` | 同步到 Agent |
| `list` | 列出已安装 |
| `upgrade` | 升级能力 |
| `uninstall` | 卸载能力 |
| `info` | 能力详情 |
| `status` | 连接状态 |

## 4. 数据流

```
采集层 (Collector)
    │
    ▼
分析层 (Analyzer) ──▶ 打包层 (Packager)
    │                       │
    ▼                       ▼
Registry (.snp) ◀────── 打包输出
    │
    ▼
安装层 (CLI) ──▶ Agent 目录
```

## 5. 支持的 Agent

| Agent | 安装路径 | 同步模式 |
|-------|---------|---------|
| Claude Code | `~/.claude/skills/` | symlink/copy |
| OpenCode | `~/.opencode/skills/` | symlink/copy |
| OpenClaw | `~/.openclaw/plugins/` | copy |

## 6. 依赖

- commander - 命令行解析
- inquirer - 交互式输入
- axios - HTTP 客户端
- yaml - YAML 配置
- fs-extra - 文件系统
- crypto-js - 加密
- chalk - 彩色输出
- semver - 版本比较
